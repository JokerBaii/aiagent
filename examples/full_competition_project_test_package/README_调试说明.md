调试说明

这个目录是一个完整竞赛项目材料测试包，适合拖入 Workbench 或作为 ProjectLoader 输入路径。

建议调试入口：

- 目录输入：examples/full_competition_project_test_package
- 压缩包输入：examples/full_competition_project_test_package.zip
- 规则目录：rules
- 建议赛道：business_innovation；不指定赛道时，平台也应根据商业计划、财务预测和市场调研自动识别为商业创新类。

预期覆盖的资产角色：

- 项目申报材料
- 商业计划书
- 市场调研
- 竞品分析
- 财务预测
- 用户调研
- 路演材料
- 部署说明
- 成果证明
- 软著证明
- 社会实践证明
- 论文或方法说明
- 实验数据
- 独立市场数据来源
- 源码
- 构建入口

这个包刻意不包含真实密钥、真实个人隐私数据、build 目录、node_modules 或 vendor 目录。
