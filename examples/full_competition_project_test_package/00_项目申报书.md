# 智农云链

项目类别：大学生创新创业竞赛商业创新类项目。

项目摘要：智农云链面向中小型农业合作社，提供农资采购协同、订单履约记录、培训服务排期和可信材料归档能力。项目材料采用可追溯证据链方式整理，便于竞赛审计、答辩复核和二次补证。

目标用户：县域中小农业合作社负责人、农技服务站和参与农产品订单履约的学生实践团队。

痛点：合作社常见问题是农资采购记录分散、订单履约凭证不统一、培训服务数据难以回溯，导致竞赛材料中的用户、收入和社会影响声明不易复核。

解决方案：建设轻量化协同平台，把采购需求、服务排期、订单回执、培训记录和证明材料统一归档，并输出可核验的项目证据目录。

产品或服务：平台包含合作社档案、农资需求单、培训活动记录、订单履约台账、证据材料清单和答辩报告导出模块。

技术路线：采用 C++ 命令行原型实现材料一致性检查，核心模块包括声明解析、证据匹配、风险规则执行和调试日志输出。

商业模式：由合作社或县域农技服务机构按年度服务包付费，学生团队提供部署、培训和材料整理服务，收入来源包括基础服务费和项目制数据整理服务费。

市场规模：华东地区中小合作社数字化服务 TAM 估算为 18.4 亿元，SAM 为 4.2 亿元，估算依据见市场调研报告和样本访谈记录。

竞品分析：竞品包括通用表单工具、农业 ERP 轻量版本和合作社自建台账，项目差异化在于竞赛材料证据链和低成本部署。

财务预测：预计 2026 年试点服务收入 12.6 万元，毛利率 46%，成本结构和测算假设见财务预测表。

团队结构：团队由 5 名成员组成，分工为产品、后端、数据整理、市场调研和答辩材料。

当前成果：已有用户包括 3 家合作社的 386 名登记成员，2026 年 4-6 月试点服务收入合计 18200 元；软著受理和合作备忘录见证明材料。

社会价值：项目服务对象覆盖 4 个村的合作社成员，累计组织 3 次培训活动，帮助合作社形成可复核的服务台账。
